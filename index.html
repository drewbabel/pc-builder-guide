Plan Your Build

Ensure CPU, motherboard, and RAM compatibility

Select Intel or AMD processor based on budget and use case requirements

2.2
Match Motherboard Socket

Ensure motherboard supports your CPU socket (LGA1700, AM5, etc.)

Select Compatible RAM

Choose DDR4 or DDR5 based on motherboard support, aim for 16-32GB

2.4
Check Power Requirements

Calculate total system power draw and add 20% headroom for PSU selection

2.5

Ensure GPU length, CPU cooler height, and PSU size fit in chosen case

Anti-static wrist strap, good lighting, and clean surface

3.1

Phillips head screwdriver, anti-static wrist strap, zip ties for cable management

3.2
Prepare Work Area

Clear, well-lit space with good ventilation. Avoid carpeted areas if possible

3.3
Organize Components

Unbox all parts, keep them in anti-static bags until needed, organize screws

3.4
Ground Yourself

Touch metal case or use anti-static wrist strap to prevent ESD damage

4

Install Components

CPU, RAM, storage, then mount motherboard in case

4.1
Install CPU

Open socket, align CPU (never force), close retention bracket. Handle only by edges

4.2
Install RAM

Open clips, align notch, press firmly until clicks. Use slots 2 & 4 for dual-channel

4.3
Install M.2 Storage

Insert at 30¬∞ angle, press down, secure with mounting screw. Remove thermal pad if needed

4.4

Apply thermal paste (rice grain size), mount cooler, connect fan cable to CPU_FAN header

4.5

Mount PSU fan-down if case has ventilation, install GPU in top PCIe x16 slot

CPUs (Processors)

The brain of your computer. Every instruction, every calculation, every decision flows through this silicon powerhouse. Understanding CPU specifications and naming is crucial for building a balanced system.

What Does a CPU Actually Do?

The Central Processing Unit (CPU) is literally the brain of your computer. It executes every single instruction your software needs - from basic math calculations to complex AI algorithms. When you click a button, open a file, or run a game, the CPU breaks down these tasks into billions of tiny operations and processes them at incredible speeds.

üèóÔ∏è CPU Architecture Explained

Cores

Think of cores as individual workers. Each core can handle one task at a time. Modern CPUs have 4-24+ cores, allowing multiple programs to run simultaneously without slowing each other down.

Threads

Hyperthreading/SMT technology lets each core pretend to be two cores. This means a 8-core CPU can appear as 16 threads to your operating system, improving multitasking performance.

Clock Speed (GHz)

How many operations per second each core can perform. Higher numbers = faster single-threaded performance. Modern CPUs boost from 3-5+ GHz depending on workload.

Cache

Super-fast memory built into the CPU. L1, L2, and L3 cache store frequently used data right next to the cores, dramatically reducing wait times for memory access.

GPUs (Graphics Cards)

Transform data into stunning visuals. From ray-traced reflections to AI-accelerated rendering, your graphics card brings worlds to life and accelerates modern computing workloads.

What Does a GPU Actually Do?

Graphics Processing Units (GPUs) are specialized processors designed for parallel computation. While CPUs excel at complex sequential tasks, GPUs handle thousands of simple calculations simultaneously. Originally designed for rendering graphics, modern GPUs accelerate everything from gaming visuals to AI training, cryptocurrency mining, and video encoding.

üé® GPU Architecture Explained

CUDA/Stream Cores

Hundreds to thousands of small processors that work in parallel. More cores = better performance for rendering, compute tasks, and higher resolutions. Think of them as a massive workforce handling simple tasks simultaneously.

VRAM (Video Memory)

Dedicated high-speed memory for textures, frame buffers, and game assets. More VRAM allows higher resolutions, better textures, and complex scenes without performance drops. Critical for 4K gaming and content creation.

RT Cores (Ray Tracing)

Specialized hardware for realistic lighting calculations. Ray tracing simulates how light actually behaves - reflections, shadows, and global illumination. Dramatically improves visual realism in supported games.

Tensor Cores (AI)

Accelerate machine learning and AI upscaling. DLSS/FSR use AI to render games at lower resolution then intelligently upscale, providing higher performance with minimal quality loss.

Technical Considerations

Thermal Resistance

Air cooling typically has 0.3-0.5¬∞C/W thermal resistance, while quality AIOs achieve 0.2-0.3¬∞C/W, meaning better heat transfer efficiency.

Thermal Mass

Liquid cooling systems have higher thermal mass, providing more stable temperatures during short bursts of high CPU load.

Noise Curves

Air coolers get dramatically louder under load, while AIOs maintain relatively consistent noise levels due to better heat dissipation.

Pump vs Fan Longevity

Quality fans last 10+ years, while AIO pumps typically last 5-7 years. Factor replacement costs into long-term planning.
